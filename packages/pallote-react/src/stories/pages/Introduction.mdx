# Pallote

Pallote React is a design system and a React UI component library. Here you will find our design guidelines, component documentation and resources on how to use and customise the library.

## Install

```shell
npm install pallote-react
```

## Dependencies

Pallote uses the `pallote-css` npm package as a CSS library for styling the components.
