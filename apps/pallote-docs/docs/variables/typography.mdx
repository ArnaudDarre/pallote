# Typography

List of the default typography variables. Since typography is at the core of every design system, having consistent font styles is paramount to make sure that users scan and consume your information well.

## Override variables

These are the default typographic style, which you can override them to match your own design system. See [Usage](/docs/getting-started/usage) for more information on how to override variables.

## List of variables

### Typeface

| Name | Value | |
| - | - | - |
| `$font` | `"Source Sans 3"` | Default body font |
| `$font-headings` | `"Source Sans 3"` | Default headings font |
| `$font-fallback` | `"Arial, sans-serif"` | Fallback font stack |
| `$font-code` | `monospace` | Font for code blocks |

### Weight

| Name | Value | |
| - | - | - |
| `$weight-regular` | `400` | Regular |
| `$weight-bold` | `700` | Bold |
| `$weight-headings-regular` | `400` | Regular weight for headings |
| `$weight-headings-bold` | `700` | Bold weight for headings |
