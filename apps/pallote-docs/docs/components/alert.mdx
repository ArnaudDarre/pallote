### Actions

Add action buttons using the `actions` prop.

<div class="docs_block">
  <Alert
    variant="notice"
    title="Update available"
    subtitle="A new version is available."
    actions={
      <>
        <Button size="sm" color="success" variant="stroke">Dismiss</Button>
        <Button size="sm" color="success">Update</Button>
      </>
    }
  />
</div>

<Tabs groupId="package" queryString>
  <TabItem value="react" label="React">

```jsx
<Alert
  variant="notice"
  title="Update available"
  subtitle="A new version is available."
  actions={
    <>
      <Button size="sm" color="success" variant="stroke">Dismiss</Button>
      <Button size="sm" color="success">Update</Button>
    </>
  }
```

  </TabItem>
  <TabItem value="css" label="CSS">

```html
      <button class="button button-sm button-success button-stroke">Dismiss</button>
```
